# Deepak Kumar Portfolio

Welcome to my professional portfolio website showcasing my work, skills, and projects.

## About This Portfolio

This is a motion-led, responsive single page portfolio built with modern web technologies like HTML, CSS, and JavaScript. The design features interactive elements such as 3D cards, animated typing effects, and smooth scroll reveals.

- Showcases my featured skills: Web Development, Java, Data Science, and more.
- Highlights selected projects with detailed descriptions.
- Includes dynamic LeetCode stats fetched live via API.
- Supports light/dark theme toggle for user preference.
- Responsive design optimized for desktop and mobile.

## Technologies Used

- HTML5
- CSS3 (including CSS Grid and Flexbox)
- JavaScript (Vanilla)
- Google Fonts (Inter)
- Font Awesome Icons

## Features

- Animated typed role display
- Responsive Projects grid with hover tilt effects
- Interactive 3D profile card
- Contact form integrated with Formspree (or your preferred backend)
- Social media follow section with icons and links
- Dark and Light mode toggle
- Live LeetCode problem-solving stats API integration
